# NDP LiDAR OOD — Project Page

This repository contains a GitHub Pages site for the paper demo:
**Neural Distribution Prior for LiDAR Out-of-Distribution Detection**.

## Where the website lives

- The website source is in `docs/`
- Entry page: `docs/index.html`
- CSS/JS/assets: `docs/assets/`

## Enable GitHub Pages

In your GitHub repo:

1. Go to **Settings → Pages**
2. Under **Build and deployment**
   - **Source**: *Deploy from a branch*
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/docs`
3. Save, then wait ~1–2 minutes for deployment

Your site will appear at:

- `https://<username>.github.io/<repo>/`

## Customize

Edit these in `docs/index.html`:

- Authors, venue/year
- Buttons: Paper PDF / arXiv / Code / Dataset
- Abstract text
- Demo videos: replace `docs/assets/demo_qualitative.mp4` and `docs/assets/demo_comparison.mp4`
- Figures: replace `docs/assets/teaser.svg`, `method.svg`, and `result_*.svg`
- BibTeX block
